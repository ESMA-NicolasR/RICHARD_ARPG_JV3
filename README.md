# RICHARD_ARPG_JV3
Student project, top down ARPG in UE5

## Controls
| Action              | Input             |
|---------------------|-------------------|
| Move to destination | Left Click        |
| Follow cursor       | Left Click (hold) |
| Cast current spell  | 1                 |
| Next spell          | Mouse wheel up    |
| Previous spell      | Mouse wheel down  |
| Speak with NPC      | Space             |

## How to play
- Move around in the arena and cast your spells.
- You can find a new spell on the ground (walk on the white box).
- Other spells can be bought from the merchant NPC (spells from the first row cost 5, those from the second row cost 10).
- Speak with the other NPC to start an enemy wave.
- Kill enemies to gain money, clear a wave before sending the next one !
- If you lose all your health, you start again from scratch.

## Retours pédagogiques
### Expérience attendue
Le jeu est une sorte d'arène sandbox où on peut détruire des ennemis à foison à l'aide de différents spells.

### Difficultés
J'ai eu du mal à intégrer l'UI du shop dans le jeu. Au début j'en avais fait une UI en 3D dans le monde mais je n'arrivais pas à modifier la taille des composants ni à cliquer correctement sur les boutons. Finalement j'en ai fait une interface classique qui s'active et se désactive.
